# QR Code Generator (UPI) 🚀

A sleek, mobile-responsive web application to generate instant UPI payment QR codes. Simply enter a UPI ID and an optional amount to create a scannable QR code compatible with all major UPI apps like **Google Pay, PhonePe, and Paytm**.

---

## ✨ Features
- **Instant Generation**: Create QR codes in real-time as you type.
- **Payment Focused**: Supports standard UPI URI protocol (`upi://pay`).
- **Mobile First**: Fully responsive design that works perfectly on smartphones and tablets.
- **Downloadable**: Save your QR code as a high-quality PNG image to share via WhatsApp, Email, or Print.
- **Minimalist UI**: Clean, distraction-free interface with a professional "PG" brand identity.

---

## 🛠️ How to Use
1. **Open the App**: Launch `QR_Code_Generator.html` in any modern web browser.
2. **Enter UPI ID**: Input the recipient's VPA (e.g., `9876543210@upi` or `username@bank`).
3. **Set Amount (Optional)**: If you want to request a specific amount, enter it in the Amount field.
4. **Generate**: Click the **"Generate Payment QR"** button.
5. **Share/Pay**: Scan the code directly from the screen or click **"Download QR Image"** to save it for later.

---

## 💻 Tech Stack
- **HTML5 & CSS3**: For structure and modern, responsive styling.
- **JavaScript (ES6)**: For dynamic URI construction and logic.
- **QRCode.js**: A lightweight library used for high-fidelity QR generation.
